# eWMS System Back-end & Database Module

## Introduction
This repo is designed to store the Back-end and MongoDB code.

## How to run the code?

Step 1:  
`npm install`  
to install the dependency packages

Step 2:  
`npx nodemon server.js`  
to run the server at `http://localhost:5001`
